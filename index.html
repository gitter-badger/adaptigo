Adaptigo is a installer for the things that Debian GNU/Linux doesn't have installed,

inspired and forked from the fedy installer for fedora

Introduction

Adaptigo lets you install multimedia codecs and additional software that Debian doesn't want to ship or have installed by default, like mp3 support, Adobe Flash, Oracle Java etc., and much more with just a few clicks.

Installation

Adaptigo can be installed with our script which automatically sets up the Some repos and dependencies. Usage

bash -c 'su -c "curl xpd259.github.io/adaptigo/web-install.sh -o adaptigo-installer && chmod +x adaptigo-installer && ./adaptigo-installer"'

After installation, search for Adaptigo in the menu/overview or type adaptigo in the terminal.
